# 🎲 Dice Rolling Game (Python)

## 📌 Project Overview

The **Dice Rolling Game** is a simple Python project that simulates rolling a dice.
Each time the user runs the program, the computer generates a random number between **1 and 6**, just like a real dice.

This project helps beginners understand:

* Random number generation in Python
* User input handling
* Loops and conditional statements
* Basic game logic
